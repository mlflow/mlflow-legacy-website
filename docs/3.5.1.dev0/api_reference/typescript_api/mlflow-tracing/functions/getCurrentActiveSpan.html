    Function getCurrentActiveSpan

    • getCurrentActiveSpan(): null | LiveSpan

      Get the current active span in the global context.

      This only works when the span is created with fluent APIs like @trace or withSpan. If a span is created with the startSpan API, it won't be attached to the global context so this function will not return it.

      Returns null | LiveSpan

      The current active span if exists, otherwise null.

      Example

      const tracedFunc = trace(() => {
        const span = getCurrentActiveSpan();
        span?.setAttribute("key", "value");
        return 0;
      });
